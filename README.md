# 📚 Data Structures & Algorithms (DSA) in C

## 📌 Overview

This repository contains my **Data Structures and Algorithms (DSA)** class assignments implemented in the **C programming language**. It covers fundamental concepts that are essential for building a strong foundation in problem-solving and efficient programming.

The repository is structured topic-wise, starting from basic data structures like arrays and progressing toward more advanced concepts such as linked lists and sorting algorithms.

---

## 🎯 Objectives

* Understand core **DSA concepts** and their implementation in C
* Develop efficient problem-solving and logical thinking skills
* Learn how data is organized, stored, and manipulated
* Prepare for coding interviews and academic evaluations
* Serve as a structured reference for revision

---

## 📚 Topics Covered

### 🔹 Arrays

* Basic operations (insertion, deletion, traversal)
* Searching techniques

### 🔹 Stack

* Implementation using arrays
* Push and Pop operations
* Applications of stack

### 🔹 Queue

* Linear queue implementation
* Basic operations (enqueue, dequeue)

### 🔹 Recursion

* Concept of recursion
* Classic problems like **Tower of Hanoi**

### 🔹 Linked Lists

* Singly Linked List (SLL)
* Circular Linked List (CLL)
* Doubly Linked List (DLL)
* Traversal and insertion/deletion operations

### 🔹 Sorting Algorithms

* Merge Sort
* Quick Sort

---

## 🛠️ How to Run the Programs

### 🔹 Using GCC Compiler

1. Install GCC
2. Open terminal or command prompt
3. Navigate to the program directory
4. Compile the code:

   ```bash
   gcc filename.c -o output
   ```
5. Run the program:

   ```bash
   ./output
   ```
